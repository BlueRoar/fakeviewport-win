fakeviewport-win setup

Configure the Chrome profile and choose your Protect grid URL.

This setup window is only for signing in, finding the exact Live View grid you want, and copying that URL into config.json. It will not start kiosk mode.

Recommended

Use your direct UNVR IP

Best for a display computer on the same LAN as the UNVR. It avoids cloud routing and usually reconnects faster.

  1. Find the UNVR IP in UniFi Network, your router DHCP leases, or on the UNVR front panel / UniFi OS console settings.
  2. Open https://YOUR_UNVR_IP/protect in this Chrome window.
  3. Accept the local certificate warning if your browser shows one.
  4. Sign in, open Protect, and choose the Live View grid you want.
  5. Copy the full address from Chrome and paste it into ProtectUrl in config.json.

Direct URL shape

Direct UNVR URLs usually look like this after you choose a dashboard or Live View grid:

https://192.168.1.100/protect/dashboard/abc123def456 https://unvr.local/protect/dashboard/abc123def456
Cloud option

Use UniFi cloud access

Use this when the display computer is not on the same local network, or when cloud access is easier for your site.

  1. Open unifi.ui.com.
  2. Sign in and select your console.
  3. Open Protect and choose the Live View grid you want.
  4. Copy the full address from Chrome and paste it into ProtectUrl in config.json.

Cloud URL shape

Cloud URLs include a long console identifier before the Protect path:

https://unifi.ui.com/consoles/ABCDEF1234567890:1234567890/protect/dashboard https://unifi.ui.com/consoles/ABCDEF1234567890:1234567890/protect/dashboard/abc123def456

Finish setup

  1. Open config.json in this folder.
  2. Paste your copied grid URL into ProtectUrl.
  3. Fill LoginUsername and LoginPassword if you want automatic login recovery.
  4. Close Chrome.
  5. Run start-fakeviewport-win.cmd.

Credentials in config.json are plain text. Keep this folder on a secured display machine and do not push a filled-in password to GitHub.

Start on Windows login

After fakeviewport-win is working, run install-startup-shortcut.cmd to make it start automatically when this Windows user signs in.

  1. install-startup-shortcut.cmd creates a shortcut named fakeviewport-win.lnk.
  2. The shortcut is placed in %APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup.
  3. The shortcut runs start-fakeviewport-win.cmd.
  4. Run uninstall-startup-shortcut.cmd to remove the startup shortcut later.
